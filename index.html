Modo IA · opcional

El modo offline siempre funciona sin internet ni costo. El modo IA es un complemento opcional que activa chat real, mejora de texto y clasificación asistida. Requiere su propia API key.

IA Gateway: agregue una o más API keys. Si la primera falla (cuota agotada, saldo bajo, error de servidor), ConciencIA prueba automáticamente la siguiente. Arrastre las tarjetas con los botones ↑ ↓ para cambiar prioridad.

Nuevo proveedor

Su key se guarda solo en este navegador. ConciencIA nunca la envía a ningún servidor propio. ¿Dónde consigo una API key?

Cómo conseguir una API key

Cada proveedor tiene su propio proceso

Anthropic Claude

  1. Ingrese a console.anthropic.com
  2. Cree cuenta · agregue saldo (mín. $5 USD)
  3. Settings → API Keys → Create Key
  4. Copie la key (empieza con sk-ant-) y péguela aquí

OpenAI

  1. Ingrese a platform.openai.com/api-keys
  2. Cree cuenta · agregue saldo (mín. $5 USD)
  3. Create new secret key
  4. Copie la key (empieza con sk-) y péguela aquí

Google Gemini

  1. Ingrese a aistudio.google.com/apikey
  2. Get API key (gratis para uso moderado)
  3. Copie la key (empieza con AIza) y péguela aquí

Importante: su key se guarda solo en este navegador (localStorage). ConciencIA es una aplicación estática sin servidor propio · las llamadas van directo browser → API. Si comparte el computador, desactive el modo IA al terminar.
